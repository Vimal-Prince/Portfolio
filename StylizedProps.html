Project Summary

This project showcases a set of stylized, game-ready props created to accompany my previously posted character. The assets were designed with clean topology and optimized for real-time rendering.

Highpoly modeling and UV layout were done in Maya, while ZBrush was used for sculpting fine details. The baking and texturing process was completed in Substance Painter, focusing on stylized surface definition and color balance. Final renders were produced in Marmoset Toolbag to highlight form, material response, and presentation quality.

Texture maps: 2K

Software Used
